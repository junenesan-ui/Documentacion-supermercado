# Supermercado Económico Plus — Portal de Documentación Operativa


Este repositorio contiene la documentación oficial de procesos, KPIs, lecciones aprendidas y formatos estandarizados del Supermercado Económico Plus. Su objetivo es centralizar y preservar el conocimiento organizacional para capacitación, auditorías y mejora continua.


## Contenido principal
- **Procesos operativos (8):** Procedimientos estandarizados con diagramas actualizados.
- **KPIs (5):** Indicadores clave con metodología de medición.
- **Lecciones aprendidas (15):** Hallazgos de la implementación y mejoras aplicadas.
- **Formatos estandarizados:** Plantillas para reportes diarios y semanales.
